# RAG_LLM_Documents
Include three steps:
1. Data Ingestion:
   - Read documents.
   - Test splitter (use rule base and Semantic Splitter).
   - Embedding model.
   - Vector database (use Weaviate).
2. Retrieval:
   - Seach type: hybrid.
   - Reranking.
   - Query Rewritting.
3. Generation:
   - Set LLMs.
   - Query Engine.
   - Use Templates for Prompts
